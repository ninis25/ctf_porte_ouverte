Le Code Source

Description

Un développeur a caché un message dans le code source de cette page. Trouvez-le !

Connexion

Indices (0/4 utilisés)

Cliquez pour déverrouiller

Les développeurs laissent souvent des commentaires dans le code HTML. Ces commentaires ne sont pas visibles sur la page mais sont présents dans le code source.

Coût: 5 points

Utilisez le clic droit de votre souris sur la page et cherchez une option pour voir le code source.

Coût: 10 points

Le raccourci Ctrl+U (ou Cmd+U sur Mac) permet d'afficher le code source de la page. Le flag est caché dans un commentaire HTML.

⚠️ Attention: Coût: 25 points

Ceci révélera la solution complète

Solution pas à pas :

  1. Faites un clic droit sur la page
  2. Cliquez sur "Afficher le code source" ou utilisez Ctrl+U
  3. Cherchez dans le code HTML le commentaire contenant : FLAG{INSPECT_ME}
  4. C'est le flag à soumettre

Outils suggérés

  • Navigateur web (Chrome, Firefox, etc.)
  • Outils de développement du navigateur (F12)
