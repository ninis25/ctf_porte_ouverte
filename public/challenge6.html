Fichier Caché

Description

Un fichier important a été caché dans l'arborescence du système. Trouvez-le et récupérez son contenu.

Terminal

Indices (0/4 utilisés)

Les fichiers cachés sous Linux commencent par un point (.). Utilisez ls -la pour les voir.

Coût: 5 points

La commande find peut vous aider à chercher des fichiers. Essayez : find . -type f -name ".*"

Coût: 10 points

Le fichier est caché dans le dossier Documents. Utilisez : cd Documents puis ls -la

⚠️ Attention: Coût: 25 points

Ceci révélera la solution complète

Solution pas à pas :

  1. cd Documents
  2. ls -la
